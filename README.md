Repo for the MSE131 Final Project - Product Management for the launch of a start-up

# Project Timeline Simulation

## What this project does
This Streamlit app simulates the launch timeline for a startup app + budgeting card.

It uses:
- precedence relationships
- CPM (critical path method)
- slack
- uncertain task durations
- Monte Carlo simulation

## Files
- app.py -> Streamlit app
- requirements.txt -> Python packages

## How to run

1. Create a folder and save the files.
2. Open a terminal in that folder.
3. Install packages:

pip install -r requirements.txt

4. Run the app:

streamlit run app.py

## Performance measures
1. Total project completion time
2. Probability of finishing by the target launch date

## Project management principles used
- forward pass
- backward pass
- critical path
- slack
- precedence constraints
- uncertain activity durations
